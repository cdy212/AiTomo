## [2026-04-23] 추가 기능 업데이트: 자동 화면 번역 및 메뉴 조정 (고도화)

### 변경 사항
- **구글 번역기 평가 팝업 제거**: 번역 시 우측 하단이나 텍스트 선택 시 나타나는 구글 평가 팝업(`goog-te-balloon-frame`, `goog-gt-tt` 등) 및 툴팁을 CSS 강제 제어로 완전히 숨김 처리
- **번역 로직 모듈화 및 로컬스토리지 연동**: 상단 번역 기능을 모든 페이지에 공통으로 적용하기 위해 `js/translateManager.js`를 신규 생성하여 스크립트 모듈화
- **기본 언어 일본어 세팅**: 사용자가 처음 접속 시 일본어로 번역되게 설정하고, 마지막으로 선택한 언어를 브라우저(LocalStorage)에 저장하여 다음 접속 시 자동으로 유지되게 개선
- **메인화면 건의 항목 숨김**: `index.html` 내 우측 '개발/운영진에게 건의' 섹션을 `hidden` 클래스로 처리하여 임시 비활성화
- **음성 인식 연동(보이스매니저)**: `voiceManager.js`가 새로 만든 번역 매니저의 현재 언어 상태를 동적으로 추적하도록 연동 구조 개선

## [2026-04-23] 메인 화면 및 네비게이션 개편

### 변경 사항
- **메인 화면 문구 수정**: "마이크로 편하게..." -> "한일부부 애기 친구분들의 편안한 모임 공간"으로 변경
- **커피라운지 링크 수정**: "커피라운지에 고민 남기기" 문구를 "커피라운지 수다떨기"로 수정하고 `qna.html`로 링크 연결
- **미구현 카테고리 숨김**: 메인 화면의 상단 카테고리 탭 버튼들 `hidden` 처리하여 임시 숨김
- **커피라운지 프리뷰 수정**: 메인 화면 리스트 데이터에서 undefined가 출력되는 문제 해결 (데이터 속성 `post.title` -> `post.content` 변경)
- **전체 메뉴 순서 및 명칭 수정**: `홈(/), 커피라운지, 꿀팁[맵], 건의, 정보교류[공사중]` 순서로 데스크탑 및 모바일 탭 바 동일하게 개편 적용
- **알림 아이콘 숨김**: 네비게이션의 종 모양(알림) 아이콘 기능이 미구현 상태이므로 `hidden` 처리하여 숨김
